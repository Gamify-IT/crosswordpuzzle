# crosswordpuzzle

This is a simple crosswordpuzzle. With given questions and answers it automatically generates a crosswordpuzzle.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
